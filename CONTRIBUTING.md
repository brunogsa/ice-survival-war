# Requirements

- Lua 5+
- LOVE 11+

## Installing Love

```shell
sudo add-apt-repository ppa:bartbes/love-stable
sudo apt-get update
sudo apt-get install -y love love-doc

# Check installation
love --version
```

## Installing Lua

```shell
sudo apt install -y lua5.3

# Check installation
lua -v
```

# Running the Game

```shell
love .
```
